# 编码类型
\u开头的是unicode编码。支持中文和英文。很容易和URL编码混淆。
\u开头的是二进制编码

# 加密类型特征
base64：base64中包含大写字母（A-Z）、小写字母（a-z）、数字0——9以及+/；一般以‘==’结尾
base32：base32中只有大写字母（A-Z）和数字234567
base16：base16中只有数字0-9以及大写字母ABCDEF
