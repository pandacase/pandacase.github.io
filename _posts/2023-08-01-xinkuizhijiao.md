回到家后开始查看学生名单，发现各年级是直接按成绩排序的，直接分班吧不是很妥。

于是就心血来潮整了个随机分班小程序给各年级分班。

```python
def group_students(stuList, numOfGroups):
    random.shuffle(stuList)  # random the list
    groups = {}
    for i, student in enumerate(stuList):
        group_number = i % numOfGroups  # set group num
        if group_number not in groups:
            groups[group_number] = []
        groups[group_number].append(student)  # add to group
    return groups


### print the result

stuList = load_data("./Grade 5.xlsx", rowStart=3, rowEnd=90, col=0)
result = group_students(stuList, 3)

for group_num, group_students in result.items():
    print(f"Group {group_num+1}:")
    for student in group_students:
        print(student)
    print()
```

后来发给大家的各班的名单，就这样诞生了。每一位班主任和自己的学生们的缘分，在此刻定格。
